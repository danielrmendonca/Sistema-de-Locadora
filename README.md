# Comandos para rodar no back

Instalar o vcpkg e o cpprest

vcpkg install cpprestsdk cpprestsdk:x64-windows

Na raiz do back rodar:

 mkdir build
 cd build

 em /build
 cmake ..

 cmake --build .
